# 🏠 Real Estate Sales Analytics Dashboard

## 📌 Project Overview

This project focuses on analyzing real estate sales data using **Power BI** to generate meaningful business insights.
The dashboard helps stakeholders understand sales performance, customer trends, and regional growth patterns.

---

## 🎯 Objectives

* Analyze overall sales performance
* Identify top-performing agents
* Track revenue across cities and regions
* Monitor discounts and negotiation trends
* Understand payment status distribution

---

## 📂 Dataset Description

The dataset follows a **Star Schema Model** with fact and dimension tables:

### 🔹 Fact Tables:

* `Fact_Sales`
* `Fact_Leasing`
* `Fact_Expenses`
* `Fact_Leads`

### 🔹 Dimension Tables:

* `Dim_Property`
* `Dim_Customer`
* `Dim_Agent`
* `Dim_Location`
* `Dim_Date`

---

## 🧠 Data Modeling

* Implemented **Star Schema**
* Created relationships (1-to-many)
* Used a **Date Table** for time intelligence
* Applied data cleaning and transformation

---

## 📊 Key KPIs

* 💰 Total Sales Revenue
* 📉 Total Discount Amount
* 📈 Average Selling Price
* ⏳ Average Negotiation Days
* 📊 Sales Growth Trend

---

## 📊 Dashboard Features

* 📅 Monthly Sales Trend Analysis
* 🌍 Region-wise & City-wise Performance
* 👨‍💼 Agent-wise Sales Contribution
* 💳 Payment Status Distribution
* 🔍 Interactive Filters (Slicers)
